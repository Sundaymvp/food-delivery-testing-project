# Food Delivery Web App Testing Project

## Description
This project demonstrates manual and API testing for a food delivery web application built with Spring Boot.

## Testing Activities
- Manual test cases for login, registration, order placement
- API testing with Postman
- Database validation using H2
- Security and role-based access testing

## Tools Used
- Java 17
- Spring Boot
- Maven
- H2 Database
- Postman
- GitHub
